NIDE — мощнейшая IDE для разработки модов для MCPE. Подсветка кода, автодополнение, упаковка модов и многое другое — всё это есть в Neat IDE! Стабильная версия программы доступна пока что только для Windows, но работа над Android NIDE (ANIDE) уже начата.
NIDE 2017, по сравнению со своим предшественником, ModPE Editor’ом, стала намного качественнее и удобнее. В этой небольшой статье я представляю Вам список основных изменений:

>Изменены название и иконка программы.

>ADB для автоматической загрузки скриптов в устройство. Решает проблему времени перемещения файла с компьютера на планшет.
>Исправлен и улучшен визуальный редактор текстур.

>Добавлен инструмент Пипетка в редакторе текстур.

>Более удобный инструмент выбора цвета в редакторе текстур.

>Добавлен инструмент Текстурирования в редакторе текстур. Имитирует неровности поверхности.

>Абсолютно новая система проектов с возможностью сбора скрипта из нескольких выходных файлов.

>Убрана отдельная конфигурация для чистого js или .modpkg, вместо этого есть один общий ModPE.

>Теперь проект — не просто папка с файлами, а упорядоченная структура, все благодаря главному файлу проекта — .nproj.

>Убрано окно User Autocomplete Items за ненадобностью и неприменимостью в реальных проектах.

>Новая система библиотек — папок с файлами lib.js, info.nlib и, по мере необходимости, другими.
Несколько предустановленных библиотек — ItemsEngine для визуального конструктора предметов, Randomizer для более гибкого получения случайных значений, Util с разными полезностями, Timers для создания таймеров.
Возможность создания пользовательских библиотек для того, чтобы делиться кодом и инкапсулировать лишнюю логику.

>Библиотека ModCore распространяется вместе с NIDE и практически полностью интегрируется при необходимости.

>Возможность запуска Javascript-кода (без ModPE, только print и clientMessage) для проверки алгоритмов перед применением.

>Добавлена динамическая проверка правильности js-кода, на данный момент исправляет ошибки только в синтаксисе.

>Консоль для вывода ошибок или отчетов.

>Два пресета цветовых тем и отдельная настройка текстового редактора.

>Визуализация подключенных библиотек.

>Сборка и отправка в устройство в один клик.

>Импорт модов в формате .modpkg.
Более удобная левая панель, новые функции в ее конекстном меню.
>Увеличен размер окна по умолчанию.

>Исправлено множество мелких багов…

Таким образом, NIDE из ModPE Editor’a превратилась в настоящую IDE для быстрого и удобного создания модификаций для игры Minecraft: PE!
