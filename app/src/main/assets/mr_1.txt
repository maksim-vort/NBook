Привет! Вижу, ты хочешь научиться создавать моды для Minecraft: Pocket Edition... Что ж, ты скачал правильное приложение.
Для разработки модов используется библиотека ModPE, использующая язык программирования JavaScript. Если вы начинаете писать без опыта программирования, то советую прочитать несколько первых уроков, посвящённым JavaScript (в будущем может быть написано просто js). 
Теперь о необходимом софте. Понадобится сама игра Minecraft: Pocket Edition, утилита под названием BlockLauncher, файловый менеджер (ES Проводник отлично подойдёт), текстовый редактор (рекомендуем попробовать наш ANIDE, ссылка на скачивание есть в разделе "О программе") и редактор изображений на уровне пикселей (Pixel Art).

Данные приложения являются необходимым минимумом для начинающего разработчика модов.
